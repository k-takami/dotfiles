# jsp-conversion-erb-tool package

## 導入方法  
tool/jsp-conversion-erb-toolをダウンロードして、~/.atom/packages/に置いてください。  

選択した範囲内の<%～%>で囲まれている文字を  
下記のパターンで置換します

## 現状の置換表
| 変換前 | 変換後 |
|---|---|
| // |  # |
| -- |  #-- |
| } else { |  else |
| } else if { | elsif |
| {  | （空白） |
| }  | end|

## あると便利なもの
| 変換前 | 変換後 | コメント |
|---|---|---|
| <%@page xxx%> | <%#@page xxx%> | JSPのpageディレクティブはコメントアウト |
| <%@include xxx%> | <%#@include xxx%> | JSPのincludeディレクティブはコメントアウト |
| &lt;jsp:useBean xxx> | <%#jsp:useBean xxx%> | JavaBeansのインスタンス化はコメントアウト |


## 使用手順
置換を行いたいところを範囲選択した状態で  ctrl＋alt＋o　と入力する 

または

置換を行いたいところを範囲選択した状態で  ctrl＋shift＋p でコマンドパレット開いて jsp conversion erb tool:toggle と入力する
